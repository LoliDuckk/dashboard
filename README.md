# Frontend Dashboard

Интерактивный дашборд для визуализации метрик с интегрированной таблицей и графиками. Проект позволяет отслеживать различные показатели (выручку, количество чеков, гостей и т.д.) с возможностью просмотра динамики по дням недели.

## Возможности

- **Интерактивная таблица** с выбором метрик
- **Динамические линейные графики** для каждой метрики с Tooltip
- **Поддержка различных единиц измерения** (рубли, чеки, гости, и др.)
- **Отзывчивый дизайн** на базе Bootstrap
- **Расчёт процентного изменения** между днями

## Отслеживаемые метрики

- Выручка (Рублей)
- Наличные расчёты (Рублей)
- Безналичные расчёты (Рублей)
- Кредитные карты (Рублей)
- Средний чек (Рублей)
- Средний гость (Рублей)
- Удаления из чека (Рублей)
- Количество чеков
- Количество гостей

## Быстрый старт

### 1. Клонирование репозитория

```bash
git clone https://github.com/LoliDuckk/dashboard
cd dashboard
```

### 2. Установка зависимостей

```bash
npm install
```

### 3. Запуск в режиме разработки

```bash
npm run dev
```

Откройте [http://localhost:5173](http://localhost:5173) в браузере.

## Доступные команды

```bash
# Запуск сервера разработки

# Сборка проекта для production
```

## Технологический стек

- **React 19** – библиотека для построения UI

- **React Bootstrap 2** – готовые компоненты UI
- **Bootstrap 5** – фреймворк стилизации
- **Recharts 3** – библиотека для создания диаграмм
- **ESLint 9** – линтер для контроля качества кода
